The Chieme Framework

A deepeer dive into the serverless backend infrastructure powering this portfolio.

S3 Bucket

Houses static assets with 99.999999999% durability. Configured for static hosting with public access disabled.

Origin Access Control

Secures the S3 origin by ensuring only CloudFront can access the bucket contents, preventing direct bypass of the CDN.

AWS Certificate Manager

Automates provisioning and renewal of SSL/TLS certificates for end-to-end encryption via HTTPS.

CloudFront (CDN)

A global edge network that caches content close to users, reducing latency and offloading traffic.

Route 53 (DNS)

Scalable cloud DNS management that routes traffic to the CloudFront distribution using Alias records.
